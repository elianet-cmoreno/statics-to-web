# Más de 840.000 personas mueren anualmente por riesgos psicosociales en el trabajo

Visualización interactiva sobre las muertes causadas por riesgos psicosociales 
en el entorno laboral, desglosadas por región mundial, tipo de enfermedad y sexo.

## 🔗 Ver gráfico interactivo

[Ver visualización](https://tuusuario.github.io/nombre-repo/riesgos-laborales)

## ¿Qué muestra?

Gráfico radial que representa las muertes por región mundial (Asia y Pacífico, 
África, Europa y Asia Central, Américas y Estados Árabes), diferenciando entre 
enfermedades cardiovasculares y trastornos mentales. Al hacer clic en cada región 
se activa el desglose por sexo mostrando el porcentaje de hombres y mujeres afectados.

## Funcionalidades

- Gráfico radial interactivo con 5 regiones mundiales
- Diferenciación visual entre enfermedades cardiovasculares y trastornos mentales
- Clic en cada región para ver el desglose por sexo
- Tooltip con datos detallados por región
- Clic fuera del gráfico para desactivar la selección

## Datos

- **Fuente:** International Labour Organization (ILO)
- **Informe:** El entorno de trabajo psicosocial: avances mundiales y vías de acción
- **Año:** 2026

## Herramientas

Desarrollado íntegramente en HTML, CSS y JavaScript con SVG nativo, sin librerías externas.
Datos estructurados y preparados en Excel.

## Contexto

Este proyecto forma parte de una serie de gráficos estáticos publicados por 
Agencia EFE reconvertidos en visualizaciones interactivas para web, con el objetivo 
de recuperar lecturas que se pierden en el formato impreso o estático.
